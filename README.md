# CheckNeck Website

Premium landing page for CheckNeck — the AI posture correction app powered by AirPods head tracking.

[![Deploy on Railway](https://railway.com/button.svg)](https://railway.com/template/ZweBpe?referralCode=xsbY2R)

## Quick Deploy

1. Click the button above, OR
2. Go to [railway.com/new](https://railway.com/new) → **Deploy from GitHub Repo** → select `checkneck-website`

## Local Development

```bash
node server.js
# Open http://localhost:8090
```

## Stack

- Pure HTML/CSS/JS — no build step
- Node.js static server with gzip & caching
- 13 programmatic SVG illustrations
- Live posture monitor (DeviceOrientation API)
- Da Vinci's Codex design system (Golden Ratio)
